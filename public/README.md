# English Master PWA

Приложение для изучения английского языка с AI-репетитором на базе Groq API.

## Установка на iPhone

1. Откройте ссылку на приложение в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выберите "На экран «Домой»"
4. Приложение будет установлено как нативное

## Настройка API ключа

Для работы AI функций (диалоги, проверка упражнений):

1. Перейдите на https://console.groq.com
2. Зарегистрируйтесь и создайте API key
3. В приложении перейдите в Настройки → API Ключ Groq
4. Вставьте ваш ключ

Бесплатный лимит: ~14 400 запросов в день.

## Функции

- 📚 6 полноценных уроков с диалогами и упражнениями
- 🃏 Карточки слов с системой SRS для запоминания
- 🤖 AI репетитор для практики диалогов
- 🏆 Система достижений и уровней
- 📊 Статистика прогресса
- 🔥 Серии дней подряд (streak)
- 🌙 Темная тема (автоматически)
- 📱 Работает офлайн (без AI функций)

## Структура

```
public/
├── index.html          # Главный HTML файл
├── manifest.json       # PWA манифест
├── sw.js              # Service Worker
└── assets/
    └── icons/         # Иконки приложения

src/
├── app.js             # Основная логика приложения
├── data/
│   └── content.js     # Уроки, слова, достижения
├── services/
│   ├── groqService.js     # AI интеграция
│   └── storageService.js  # Локальное хранилище
└── styles/
    └── main.css       # iOS-стиль CSS
```

## Развертывание на GitHub Pages

1. Запушьте код в репозиторий GitHub
2. Включите GitHub Pages в настройках репозитория
3. Выберите ветку `main` и папку `/public`
4. Приложение будет доступно по ссылке

Или используйте автоматический деплой через GitHub Actions (workflow уже настроен).

## Технологии

- Vanilla JavaScript (ES6+)
- IndexedDB для хранения данных
- Groq API (Llama 3.1 70B)
- PWA (Progressive Web App)
- iOS-стиль дизайн
