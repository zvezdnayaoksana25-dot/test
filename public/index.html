Настройка AI репетитора

Для работы AI-функций (диалоги, проверка упражнений, объяснения) необходим API ключ Groq. Ключ сохраняется только на вашем устройстве.

Как получить ключ:

  1. Перейдите на console.groq.com
  2. Зарегистрируйтесь или войдите
  3. Создайте API key в разделе "API Keys"
  4. Скопируйте и вставьте ключ выше

💡 Бесплатный лимит: ~14 400 запросов в день
